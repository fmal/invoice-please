Invoice please!
==============

  > _Make your invoices so easily._

## Installation

#### Global

    npm install -g bower grunt-cli

#### Local

    npm install && bower install

### Datas

You must modify `src/datas/index.json` to insert your own stuff.

## Runtime

#### Development

    grunt

#### Distribute

    grunt dist

And that's all! Enjoy your invoice in `dist/invoice.pdf`
